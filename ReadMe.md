# Dependency
```
sudo apt install nlohmann-json3-dev
c++17
```

# Build
```
Cmake -S . -B build
Cmake --build build
```

# Run
```
./build/ATMController
```

# Usage
Starting ATM is needed
 so press 1 to start ATM

 ## ATM Menu
    1. Insert Card
    2. Enter PIN
    3. Deposit
    4. Withdraw
    5. Check Balance
    6. Exit
